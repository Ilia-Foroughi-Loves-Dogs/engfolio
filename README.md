# EngFolio

EngFolio is a React-based portfolio helper for engineering students. It turns the details of a technical project into clear, professional career materials that can be used across resumes, GitHub repositories, LinkedIn posts, portfolio pages, and interview preparation.

## Why This Project Is Useful

Engineering students often build meaningful projects but struggle to explain them in a way that is concise, credible, and career-ready. EngFolio helps bridge that gap by guiding students through the most important project details: the problem, tools used, what they built, the result, and the technical challenges they solved.

Instead of starting from a blank page, students can enter structured project information and generate polished outputs that are easier to refine, share, and discuss with recruiters, hiring managers, professors, and peers.

## Built for Engineering Career Materials

EngFolio helps engineering students turn technical projects into:

- Resume bullet points
- GitHub README summaries
- LinkedIn project posts
- Interview talking points
- Portfolio project cards

The goal is to make technical work easier to communicate without exaggerating the scope or impact of the project.

## Features

- Structured project input form for engineering projects
- Generated resume bullets based on real project details
- GitHub README summary generation
- LinkedIn project post generation
- Interview talking point prompts
- Portfolio card summary output
- Copy-ready combined output
- Local project saving with browser `localStorage`
- Ability to reload and delete saved projects
- No paid API key required

## How to Run Locally

Clone the repository and install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run linting:

```bash
npm run lint
```

## Example Use Case

An engineering student builds a solar-powered campus weather station using sensors, Arduino, Python, and a small dashboard. They can enter the project title, engineering field, problem solved, technologies used, what they built, measurable impact, hardest challenge, and project links.

EngFolio then generates reusable material such as:

- Resume bullets describing the design, development, testing, and impact
- A GitHub README summary for the repository
- A LinkedIn post explaining the project professionally
- Interview talking points for discussing technical decisions and tradeoffs
- A concise portfolio project card

## Project Purpose

EngFolio is designed to help engineering students communicate the value of their technical work. Many students complete strong class projects, capstones, prototypes, research tools, and personal builds, but those projects often remain under-explained on resumes and portfolios.

This project focuses on turning engineering work into practical career assets: resume language, GitHub documentation, LinkedIn content, and interview preparation material. By helping students describe what they built, why it mattered, and how they solved technical problems, EngFolio makes project-based experience easier to present professionally.
